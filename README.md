# 【 Invoice App 】請款單記帳列表
## [DEMO](https://chanze7995.github.io/Invoice_App/)

## 簡介

以 Vue3.js 練習製作請款單記帳列表，練習使用 Firebase。

## 網頁架構

- 請款單列表
    - 依狀態做列表排序
    - 新增請款單
- 請款單詳細資訊
    - 刪除、修改資訊與狀態

## 使用技術

- **Vue.js, Vuex, Vue Router, Vue CLI**
- **JavaScript ( ES6+ )**
- **SCSS**
- **Firestore Database**
